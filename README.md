# Couplee v4 Exact UI

画像の6画面ショーケースに寄せた、1ファイル完結版のカップルアプリです。

## 実装機能

- 初回入力：名前、アイコン、交際開始日、記念日名、関係フェーズ
- ホーム：記念日カウントダウン、今日の質問、回答開示、気分チェック、ポイント、レベル、予定
- 記念日：次の記念日、軌跡、バッジ、通知テスト、デート提案
- アルバム：写真アップロード、日付、場所、キャプション、削除
- お願い掲示板：追加、未対応、受付中、完了、ポイント加算、削除
- 共有カレンダー：月間カレンダー、日付選択、予定追加、ToDo追加、完了チェック
- プライバシー：位置情報ON/OFF、常時共有、時間限定、緊急時のみ、同意チェック、招待リンク、データ書き出し、連携解除、全削除

## 反映方法

GitHubの `sebasuchan2212/couplee-app` で既存ファイルを削除または上書きし、`index.html`, `vercel.json`, `README.md` をアップロードしてください。

今回はService Workerを使っていないため、古いキャッシュが残りにくい構成です。
