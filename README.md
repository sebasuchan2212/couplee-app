# Couplee - Couples OS

Coupleeは、記念日、アルバム、お願い掲示板、共有カレンダー、プライバシー/GPS設定をひとつにまとめたカップル向けWebアプリです。

画像のUIコンセプトに合わせて、以下の機能を実装しています。

## 実装済み機能

### 初回オンボーディング

- あなたの名前
- パートナーの名前
- ふたりのアイコン
- 交際開始日 / 大切な日
- 記念日名
- 関係フェーズ
- 遠距離設定
- ふたりの目標

入力内容はホーム、記念日、お願い掲示板、共有カレンダー、プライバシー画面に反映されます。

### ホーム

- 記念日カウントダウン
- 今日の質問
- ふたりの回答入力
- 双方回答後の回答開示
- 気分チェック
- 連続記録
- ハートポイント
- カップルレベル
- 成長マスコット
- 今日の予定
- クイック操作

### 記念日

- 次の記念日までの日数
- 交際日数
- 記念日リマインダー設定
- ブラウザ通知テスト
- ふたりの軌跡追加
- マイルストーンバッジ
- おすすめデートプラン

### アルバム

- 写真アップロード
- 思い出タイトル
- 日付
- 場所
- キャプション
- タイムライン表示
- 削除
- Memory Graph風の整理

### お願い掲示板

- お願い追加
- 投稿者設定
- 詳細文
- 報酬ポイント
- 未対応 / 受付中 / 完了ステータス
- ステータス別フィルター
- 完了時のハートポイント加算
- 協力度表示
- 削除

### 共有カレンダー

- 月間カレンダー
- 日付選択
- 予定追加
- 時間設定
- メモ
- 予定削除
- ToDo追加
- 担当者設定
- 完了チェック
- メンタルロード可視化

### プライバシー / GPS

- 位置情報共有ON/OFF
- 常に共有
- 時間限定共有
- 緊急時のみ
- 双方同意チェック
- 共有履歴
- 招待リンク作成
- データ書き出し
- パートナー連携解除
- 全データ削除

## 技術構成

- HTML / CSS / Vanilla JavaScript
- フロントエンド完結
- localStorage保存
- PWA対応
- Vercelでそのまま公開可能

## 注意

現時点では、サーバー・ログイン・クラウド同期は未接続です。
本番アプリ化する場合は、Supabase/Firebase、認証、パートナー招待、写真ストレージ、プッシュ通知、課金、E2EEを追加してください。

## Vercel公開設定

- Framework Preset: Other
- Build Command: 空欄
- Output Directory: `.`
