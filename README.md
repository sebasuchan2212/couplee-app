# Couplee v5.2 - Screen Switch UI

## 変更点

- 初期設定の「関係フェーズ」を追加
  - 交際初期
  - 近距離
  - 遠距離
  - ラブラブ
  - 安定期
  - 倦怠期ぎみ
  - 同棲
  - 新婚
  - 夫婦
  - 子あり前後
  - 忙しいカップル
  - 復縁・修復中
  - 改善したい

- パートナー連携画面を追加
  - プライバシー画面に「パートナー連携」カードを追加
  - 招待リンクに初期設定情報を埋め込み
  - 相手が招待リンクを開くと、名前・記念日・関係フェーズが反映されます

## 重要

このv5.2はフロントエンド完結版です。
招待リンクは「初期設定情報の共有」まで対応しています。

リアルタイムで相手のスマホと同期するには、次のバックエンド実装が必要です。

- SupabaseまたはFirebase認証
- couple_idによるペア管理
- 招待コード発行
- DB同期
- 写真ストレージ
- プッシュ通知
- Row Level Security / Firestore Rules
